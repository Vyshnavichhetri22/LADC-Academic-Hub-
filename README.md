# 🎓 LADC Academic Hub

LADC Academic Hub is a web application that helps students and teachers manage academic resources in one place.

## 🚀 Features

- 📚 Notes Upload & Download
- ⭐ Important Questions
- 📢 Announcements
- 📄 Past Question Papers
- 📂 Project Repository
- 📖 Syllabus
- 💼 Career Resources
- 📑 Resume Templates
- 🔐 User Authentication (Students & Teachers)

## 🛠️ Tech Stack

### Backend
- Node.js
- Express.js
- MongoDB
- Mongoose

### Frontend
- Next.js
- React.js
